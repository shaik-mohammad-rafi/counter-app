Counter App

A simple counter application built using HTML, CSS, and JavaScript. It demonstrates basic DOM manipulation and event handling, allowing users to increase, decrease, and reset the count through a clean and responsive interface.

Features:
Increase, decrease, and reset the counter
Minimal and responsive layout
Beginner-friendly JavaScript logic

Technologies Used:
HTML
CSS
JavaScript

How to Run:
Download or clone the repository
Open index.html in any web browser
No additional setup required

Project Structure
index.html
style.css
script.js

Future Improvements:
Add animations for button clicks
Add dark/light mode
Add local storage to save the counter state
